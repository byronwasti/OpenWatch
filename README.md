# OpenWatch

Open source smart watch.

## Design

1. Screen (SPI + Enable)  
    - Name: LS010B7DH01
    - digikey: 425-2899-ND
2. Microcontroller 
    - Name: STM32F303RCT6
    - digikey: 497-13304-ND
3. IMU (SPI)
4. 4 Capacitive touch sensors (4 I/O)
    - Name: AT42QT1070-SSU 
    - digikey: AT42QT1070-SSURCT-ND 
5. LiPo battery
6. Bluetooth (SPI)
    - Name: BM71BLES1FC2-0002AA
    - digikey: BM71BLES1FC2-0002AA-ND 
7. External flash (SPI)
